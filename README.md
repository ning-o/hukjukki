# 메랜 흑적기 (Beta)
**메이플랜드 게임 사용자를 위한 사냥 효율 계산기 및 버프 매니저**

비전공자로서 파이썬과 웹 프론트엔드 기술을 학습하며, 제가 실제로 플레이중인 게임에서 발생하는 불편함을 기술적으로 해결하기 위해 시작한 프로젝트입니다.
현재 **Beta 버전**으로 실전 테스트 및 기능 개선을 진행 중입니다.

## 바로가기
[프로젝트 실행 링크](https://ning-o.github.io/hukjukki/)

## 주요 기능
- **실시간 사냥 정산 (Report):** 1시간당 흑자/적자 계산에 용이
- **스마트 버프 타이머:** 다양한 직업군의 버프 지속 시간을 관리하며, 종료 전 사전 알람 기능 제공.
- **버프 상황판 (Document PIP):** 브라우저의 PIP 기술을 활용하여 게임 화면 위에서 버프 시간 확인 가능.

## 기술 스택
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend:** Python (데이터 로직 설계 및 기초 분석용)
- **Deployment:** GitHub Pages

## 기술적 난관 및 해결 과정 (Troubleshooting)
### 1. PIP 모드 내 동영상 UI 간섭 문제
- **문제:** 기존 비디오 스트림 방식 사용 시 브라우저 강제 UI(재생 바 등)가 화면을 가리는 현상 발생.
- **해결:** `Document Picture-in-Picture` API를 연구하여, 비디오가 아닌 웹 요소 자체를 팝업 창으로 분리하는 방식을 채택해 순수 데이터만 노출 성공.
